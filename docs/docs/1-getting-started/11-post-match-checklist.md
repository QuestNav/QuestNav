# Post-Match Checklist

Run through this checklist after every match to put the headset in a safe state, preserve battery, and keep everything charged for the next match.

## After Each Match

- [ ] **Put the headset in standby** — tap the Quest power button. The white LED on the front should turn **off**, and the USB-C LED should stay **green**.
- [ ] **Check the USB battery level** — read the battery bank's power meter and swap or recharge it if it's running low.

:::tip
Standby keeps the Quest ready to wake quickly for the next match while saving power. A green USB-C LED confirms the headset is still receiving power from the battery bank.
:::

## Last Match of the Day

- [ ] **Remove the headset and battery bank** from the robot.
- [ ] **Plug both in to charge overnight** so they're ready for the next competition day.

:::warning
Don't leave the headset running on the robot overnight. Removing it also protects it from accidental damage in the pits.
:::
